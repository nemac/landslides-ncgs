Data Access

DOWNLOAD DATA VIEW METADATA

The data used in this project are available for download or for use as web services from NC OneMap.

All electronic and/or hard copy products (maps, data, text, etc.) produced by this landslide hazard mapping program are considered public information and may be freely redistributed and/or copied. These products, however, are intended to serve for general planning purposes only, and are provided on an "as is" basis. These maps and other products are not a substitute for an on-the-ground site assessment by qualified individuals.
